# QuranFirst - تطبيق القرآن الكريم

تطبيق أندرويد لقراءة القرآن الكريم بخط المصحف العثماني (QCF4).

## ملاحظة مهمة - الخطوط (Fonts)

الخطوط الكبيرة (QCF4 TTF) غير مضمنة في المشروع بسبب حجمها الكبير (+100MB).

### كيفية إضافة الخطوط:
1. حمل الخطوط من مصدر QCF4
2. ضع الخطوط في: `app/src/main/assets/fonts/`

## المميزات
- عرض القرآن الكريم كاملاً (114 سورة / 604 صفحة)
- خط المصحف العثماني QCF4
- واجهة احترافية وسهلة الاستخدام
- عرض أوقات الصلاة

## البناء
```
./gradlew assembleDebug
```
